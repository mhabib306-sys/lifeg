Privacy Policy

Last updated: February 2026

Overview

Homebase ("the App") is a personal task management and life tracking application. Your privacy is important to us. This policy explains what data is collected, how it is used, and your rights.

Data Collection

  • Authentication: We use Google Sign-In via Firebase Authentication. Your email address and display name are stored to identify your account.
  • Task and Tracking Data: All your tasks, habits, scores, and tracking data are stored locally on your device and optionally synced to your private GitHub repository via the GitHub API.
  • Weather Data: If you enable the weather widget, your approximate location (city-level) is used to fetch weather data from Open-Meteo. Location data is cached locally and never sent to our servers.
  • Health Data: If you connect WHOOP or Libre integrations, health metrics are fetched via your personal API keys and stored locally.

Data Storage

Your data is stored in three places:

  • Device localStorage: All app data is stored locally in your browser/app storage.
  • GitHub (optional): If you configure a GitHub token, data syncs to a data.json file in your private repository.
  • Firebase Authentication: Only authentication credentials (email, display name) are stored in Firebase.

Third-Party Services

  • Firebase Authentication (Google)
  • GitHub API (optional cloud sync)
  • Open-Meteo (weather data, no API key required)
  • Google Calendar API (optional integration)
  • WHOOP API (optional health integration)

Data Sharing

We do not sell, share, or transfer your personal data to any third parties for advertising or marketing purposes. Data is only sent to the third-party services listed above as necessary for the features you enable.

Your Rights

  • Access: All your data is visible within the app and can be exported at any time via Settings > Export Data.
  • Deletion: You can delete your account and all associated data from Settings > Delete Account.
  • Portability: Export your data as JSON from the Settings tab.

Children's Privacy

The App is not intended for children under 13. We do not knowingly collect data from children.

Changes

We may update this policy from time to time. Changes will be reflected in the "Last updated" date above.

Contact
